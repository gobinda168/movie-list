# Movie Search App
Technologies use: ReactJs, Redux, Thunk, Typscipt, Emotion with Styled Component
## Functionalities Implemented
- Sidebar
- Navbar
- Mobile Responsiveness
- Movie Page
- Favourite List page with Localstorage
- Pagination implemented
